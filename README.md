---

## 🎯 Propósito del Proyecto

Este repositorio tiene como objetivo fortalecer el aprendizaje de **bases de datos relacionales**, mediante la práctica de:

- Creación de bases de datos
- Diseño de tablas
- Consultas SQL
- Subconsultas
- Vistas
- Procedimientos almacenados
- Triggers
- Manipulación de datos

El enfoque es 100% educativo, ideal para estudiantes que están iniciando o reforzando conceptos de bases de datos.

---

## 🧠 Contenidos

Dentro del repositorio encontrarás ejercicios organizados en las siguientes áreas:

- 📌 Estructura de bases de datos  
- 🔍 Consultas simples y avanzadas  
- 🔗 Relaciones entre tablas  
- 📊 Subconsultas  
- 🧾 Vistas  
- ⚙️ Procedimientos almacenados  
- 🔄 Triggers

## ▶️ Ejecución del Proyecto

Para ejecutar los scripts del repositorio:

1. Abre tu gestor de base de datos (MySQL, SQL Server o similar)
2. Importa o ejecuta los archivos SQL en el siguiente orden:
   - tablas/
   - inserciones (si aplica)
   - consultas/
   - vistas/
   - procedimientos/
   - triggers/

---

## 📌 Recomendaciones de uso

- Ejecuta los scripts paso a paso  
- Revisa las relaciones entre tablas antes de modificar  
- Practica las consultas manualmente para mejorar comprensión  
- No ejecutes todos los scripts sin entenderlos
